# algo-trading

This repository is dedicated to researching and backtesting algorithmic trading strategies using a high-performance, vectorized approach.

## Key Technologies

*   **Backtesting:** [VectorBT](https://vectorbt.dev/) for rapid, vectorized analysis.
*   **Environment:** Python 3.x managed via a **VSCode Dev Container** for a reproducible setup.

## Getting Started

1.  **Prerequisites:** Ensure you have [Docker Desktop](https://www.docker.com/products/docker-desktop) and the **Dev Containers** extension installed in VSCode.
2.  **Launch:** Open the folder in VSCode and select **"Reopen in Container"** when prompted.
3.  **Notebooks:** All research and experimentation are conducted within the Jupyter notebooks located in the `notebooks/` directory.
